# Expense Tracker (Python Tkinter)

A simple and user-friendly **Expense Tracker desktop application** built using **Python, Tkinter, and SQLite**.  
This application helps users record, manage, and track daily expenses through an easy graphical interface.

---

## Features

- Add new expenses
- View saved expenses
- Edit existing records
- Delete single expense
- Delete all expenses
- Store data using SQLite database
- Clean and modern UI design
- Simple and easy to use

---

## Technologies Used

- Python
- Tkinter (GUI)
- SQLite (Database)
- tkcalendar (Date Picker)

## Installation

1. Clone the repository

``` https://github.com/your-username/Expense-Tracker.git```

2. Navigate to project folder


```cd Expense-Tracker```


3. Create virtual environment


```python -m venv venv```


4. Activate virtual environment

Windows:

```venv\Scripts\activate```

5. Install required packages


```pip install -r requirements.txt```


---

## Run the Application


```python main.py```


---

## Requirements


tkcalendar


Install manually if needed:


```pip install tkcalendar```
